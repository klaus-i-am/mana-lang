Mana

A readable C++-inspired language for game development

Install Mana

Run the following in PowerShell:

irm https://mana-lang.dev/install.ps1 | iex
Note: You may need to enable script execution first:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Getting Started

  1. After installation, open a new terminal for PATH changes to take effect
  2. Verify the installation: mana --version
  3. Create your first program and save it as hello.mana
  4. Compile it: mana hello.mana -o hello.cpp
  5. Build the C++ output with your compiler (g++, clang++, or MSVC)

Prerequisites

Mana compiles to C++17, so you'll need a C++ compiler:

  • Windows: Visual Studio 2022 with C++ workload
  • macOS: Xcode Command Line Tools (xcode-select --install)
  • Linux: g++ or clang++
